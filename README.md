# Akshara-Deepa Tutor

Akshara-Deepa Tutor is an AI-assisted offline learning companion designed for SSLC (10th-grade) students. The application helps students track syllabus completion, take self-assessment quizzes, analyze subject-wise strengths, and identify learning gaps through progress analytics and personalized insights.

---

## Features

- Offline-first educational support
- Chapter-wise self-assessment quizzes
- Syllabus completion tracking
- Subject strength and weakness analysis
- Personalized learning insights
- Simple and user-friendly Android interface

---

## Technologies Used

- Kotlin
- Android Studio
- XML
- Room Database
- Jetpack Components
- Git & GitHub

---

## Purpose of the Project

The project is designed to support students in rural and underserved areas who may lack access to guided self-learning systems and continuous academic support. By providing offline educational tools and progress monitoring, the app encourages structured and effective self-study.
